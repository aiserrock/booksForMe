#SQL
Список литературы
1. Дейт К.Дж. Введение в системы баз данных, 8-е издание: Пер. с англ.: – М.: Издательский дом «Вильямс», 2006. – 1328 с.

Книга одного из основоположников теории реляционных баз данных считается обязательным учебником для изучающих эту теорию. Надо отметить, что с каждым новым изданием автор существенно перерабатывает текст, стараясь учесть новые направления развития баз данных.

2. Зафиевский А.В., Короткин А.А., Лататуев А.Н. Базы данных: Учебное пособие. – Ярославль: ЯрГУ, 2012. – 164 с.

Интернет-ресурс: http://www.lib.uniyar.ac.ru/edocs/iuni/20120401.pdf .

Пособие написано работающими в ЯрГУ авторами, вследствие чего изложение максимально приближено к читающимся на факультете ИВТ курсам.

3. Пушников А.Ю. Введение в системы управления базами данных.

Интернет-ресурс: http://citforum.ru/database/dblearn/index.shtml .

Очень хорошее изложение основ теории реляционных баз данных. Несмотря на то, что соответствующий учебник был издан в 1999 г., основы теории с тех пор не изменились.

4. Коннолли Т., Бегг К. Базы данных. Проектирование, реализация и сопро­вож­дение. Теория и практика. 3-е издание.: Пер. с англ. – М.: Издательский дом «Вильямс», 2003. – 1440 с.: ил.

Фундаментальный труд преподавателей университета Пейсли (Шотландия). Читается легче, чем учебник К.Дейта, и охватывает большинство новых направлений в технологии баз данных. В 2014 году на английском языке вышло 6-е издание этой книги: «Database Systems: A Practical Approach to Design, Implementation, and Management».

5. Марков А.С., Лисовский К.Ю. Базы данных: Учебник. – М.: Финансы и статистика, 2004. – 512 с.: ил.

Несмотря на то, что этот учебник рекомендован Министерством образования, в основной части он излишне математизирован. Но философскую и постановочную часть в нем стоит прочитать обязательно, хотя бы для того, чтобы лишний раз убедиться, что истины в последней инстанции не бывает, и свою голову на плечах иметь весьма полезно.

6. Джеймс Р. Грофф, Пол Н. Вайнберг, Эндрю Дж. Оппель. SQL : Полное руководство, 3-е издание.: Пер. с англ. – М.: Издательский дом «Вильямс», 2015. – 960 с.

Еще один фундаментальный труд: по языку SQL – основе современных баз данных. Стремится охватить все особенности этого языка, включая особенности реализации в различных системах управления базами данных. Рекомендуется хотя бы к однократному просмотру.

7. Грабер М. SQL . – Издательство «Лори», 2007. – 672 с.

Аналогичная книга, не столь фундаментальная, но тоже приемлемая для изучения и использования языка SQL .

8. Кригель А., Трухнов Б. SQL. Библия пользователя, 2-е издание.: Пер. с англ. – М.: ООО «И.Д.Вильямс», 2010. – 752 с.

Особенностью этой книги является описание реализации языка SQL в различных СУБД: Oracle, DB2 и SQL Server.

9. Кляйн К., Кляйн Д., Хант Б. SQL. Справочник, 3-е издание. – Издательство «Символ-Плюс», 2009. – 656 с.

10. Silberschatz A., Korth H.F., Sudarshan S., Database System Concepts, 6-th ed.– New York: McGraw-Hill, 2010. – 1376 p.

11. Нестеров С.А. Базы данных: учебник и практикум для академического бакалавриата. - М.: Юрайт, 2017. – 230 c.

12. Базы данных и СУБД: методические указания. Сост. А. В. Зафиевский. - Ярославль: Яросл. гос. ун-т, 2008. – 47 c.

13. С.Д. Кузнецов. Основы баз данных. Учебное пособие. Интернет-Университет Информационных Технологий. Москва, 2007.

14. А. Бьюли. Изучаем SQL. Санкт-Петербург - Москва: Издательство «Символ-Плюс», 2016. - 312 с.

Книга не только позволит приобрести начальные знания, но и расскажет о наиболее часто употребляемых мощных средствах языка SQL, используемых опытными программистами.

15. Э. Молинаро. SQL. Сборник рецептов. Санкт-Петербург - Москва: Издательство «Символ-Плюс», 2009. - 672 с.

Книга предназначена тем, кто уже знаком с основами языка запросов SQL и хочет повысить свой профессиональный уровень. Она будет полезна и экспертам SQL, поскольку автор предлагает варианты решения задач для разных СУБД: DB2, Oracle, PostgreSQL, MySQL и SQL Server. Если вы постоянно работаете с SQL на одной платформе, то, возможно, найдете в рецептах более эффективное решение на другой. Вы научитесь использовать SQL для решения более широкого спектра задач - от операций внутри баз данных до передачи данных по сети в приложения.

16. http://www.sql.ru

В список не включены книги по конкретным системам управления базами данных, часто также содержащим основы теории. Таких книг, особенно по системам Oracle , Microsoft SQL Server , MySQL , очень много, и значительную их часть можно найти в сети Интернет. При практической работе они обязательно понадобятся (хотя лучший вариант – читать сопутствующую документацию), но предварительно все-таки надо изучить основы теории, хотя бы для того, чтобы знать, чего в этих книгах не хватает.
